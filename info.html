SAND

SAND is a generative image filter that creates digitized textures from a 2D image input. Geometric glyphs are used to convert tonal values into a corresponding typographic threshold, while variables such as font-size, color, the threshold order of glyphs, and the input image are randomized in order to generate unique combinations and textures. The randomizing of font-size creates uneven line widths that when staggered evoke the smearing or blurring aesthetic quality of sand art, which informed the title for this project.


My initial research for this project began with textiles and knitting patterns and exploring how to convert these visual qualities into a digital space. I was inspired by the work of Andreas Gysin and Julian Hespenheide, and referenced The Coding Train for an ASCII image filter tutorial to start my code.
